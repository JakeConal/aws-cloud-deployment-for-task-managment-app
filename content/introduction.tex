\section{Giới thiệu}

\subsection{Phạm vi ứng dụng}

Ứng dụng Task Manager hỗ trợ các chức năng chính sau:

\begin{itemize}
    \item Đăng ký và đăng nhập người dùng thông qua Amazon Cognito.
    \item Tạo, xem, cập nhật và xóa công việc.
    \item Lưu trữ dữ liệu công việc bền vững trong Amazon DynamoDB.
    \item Chỉ cho phép người dùng đã xác thực gọi các API \texttt{/tasks}.
\end{itemize}

\subsection{Tóm tắt triển khai}

Các điểm triển khai chính của hệ thống:

\begin{itemize}
    \item Frontend được lưu trong S3 bucket riêng tư và chỉ được phục vụ qua CloudFront.
    \item CloudFront sử dụng Origin Access Control (OAC) để đọc object từ S3.
    \item Backend sử dụng API Gateway REST API stage \texttt{prod} và bốn Lambda function riêng biệt.
    \item Runtime của Lambda là Node.js 20.x.
    \item Lambda được đặt trong private subnet của custom VPC \texttt{TaskManagerVPC}.
    \item Lambda truy cập DynamoDB thông qua DynamoDB Gateway VPC Endpoint.
    \item API được bảo vệ bằng Cognito Authorizer.
    \item IAM được cấu hình theo nguyên tắc least privilege với bốn role riêng cho bốn Lambda function.
    \item CloudWatch Dashboard, CloudWatch Alarms, SNS và AWS Budget được cấu hình để giám sát và kiểm soát chi phí.
\end{itemize}
